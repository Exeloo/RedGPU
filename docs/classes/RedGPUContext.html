Class RedGPUContext

RedGPU.initialize 실행이후 생성 제공되는 객체.

  • WebGPU 초기화시 얻어낸 기본 정보들을 속성으로 가진다
  • View 객체를 소유하며 실제 최상위 컨테이너 역활을 한다.
